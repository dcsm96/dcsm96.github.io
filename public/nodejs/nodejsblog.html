Core features

A few helpful utilities

MongoDB

The mongodb & monk dependencies let us to connect easily with a mongo database hosted in mongolab.

Validator

I've included a dependency to validate the forms, by using express-validator and confirm that some important fields in the form aren't empty, or even if is an email field.

Flash Messages

When supplied valid credentials, and incorrect, the connect-flash dependencie help us with a few lines of code to display alerts, Depending on the case, it can be a successful or warning alert.

Multer

Multer let us handle multipart/form-data, in this case for upload and encrypt the images of the posts.

Moment

With Moment we can format the date's post storaged in the Mongo database and finally display it.
